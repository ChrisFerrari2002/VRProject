\doxysubsection{Detailed Description}
\doxylink{class_board}{Board} class header file. 

This file contains the definition of the \doxylink{class_board}{Board} class that represents the chessboard.

The \doxylink{class_board}{Board} class manages the state of the chessboard, including the pieces on the board, the cemetery for captured pieces, and the history of moves for undo/redo functionality.
