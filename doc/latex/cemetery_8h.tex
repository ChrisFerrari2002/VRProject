\doxylink{class_cemetery}{Cemetery} class header file. 

This file contains the definition of the \doxylink{class_cemetery}{Cemetery} class that represents the cemetery for captured chess pieces.

The \doxylink{class_cemetery}{Cemetery} class manages the captured pieces, keeping track of their positions and the number of pieces eliminated for each color.
