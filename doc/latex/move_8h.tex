\doxysubsection{Detailed Description}
\doxylink{class_move}{Move} class header file. 

This file contains the definition of the \doxylink{class_move}{Move} class that represents a move made on the chessboard.

The \doxylink{class_move}{Move} class stores information about a move, including the piece moved, the start and end positions, and whether the piece was captured.
